# MoM Repository – Cloud Community Club

This repository stores all Minutes of Meetings (MoM).

## Structure

- `templates/` → MoM template  
- `2026/` → Year-wise meetings  

## Naming Convention

Use only the date:
YYYY-MM-DD.md  

**Examples:**
- 2026-03-26.md  
- 2026-04-02.md  

## Purpose

- Track discussions  
- Record decisions  
- Assign tasks
